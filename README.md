# vigilant-spork

## Numerology Calculator Web App

This repository contains a numerology calculator web app that allows users to input their name and birthdate to calculate various numerology numbers and their meanings.

### Features

- User interface to input the user's name and birthdate
- Calculation of the user's life path number based on their birthdate
- Calculation of the user's expression number based on their name
- Calculation of the user's soul urge number based on the vowels in their name
- Calculation of the user's personality number based on the consonants in their name
- Display of the calculated numerology numbers and their meanings
- Responsive design to handle different screen sizes
- Handling of invalid input with user-friendly error messages

### Usage

1. Open the `index.html` file in a web browser.
2. Enter your name and birthdate in the input fields.
3. Click the "Calculate" button to trigger the numerology calculations.
4. The calculated life path number, expression number, soul urge number, and personality number along with their meanings will be displayed.

### Files

- `index.html`: The main HTML file containing the user interface.
- `styles.css`: The CSS file for styling the user interface.
- `script.js`: The JavaScript file for implementing the numerology calculations and handling user interactions.
- `meanings.json`: The JSON file containing the meanings of the numerology numbers.

### Technologies Used

- HTML5 for structuring the user interface
- CSS3 for styling the user interface
- JavaScript for implementing the numerology calculations and handling user interactions
- JSON for storing the meanings of the numerology numbers
- Fetch API for loading the `meanings.json` file
- Responsive design techniques for handling different screen sizes
